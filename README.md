# Korbin Hillan Linktree Website

This project is a personal "Linktree" style website for Korbin Hillan, designed to showcase links to various platforms, a portfolio, a resume, and a form submission feature. The site is built with modern web technologies and includes engaging design elements like particle effects and responsive layouts.

## Features

- **Responsive Design**: The website adjusts to various screen sizes, ensuring a smooth user experience on desktop and mobile devices.
- **Particle Background**: A dynamic particle effect background powered by `particles.js`, enhancing visual appeal.
- **Custom Link Buttons**: Interactive buttons linking to Korbin's portfolio, LinkedIn, GitHub, and other platforms.
- **Form Submission**: An embedded form for user inquiries, processed dynamically without leaving the site.
- **Accessibility**: The site uses clear typography, proper alt attributes for images, and consistent navigation to support accessibility.

## Technologies Used

- **HTML5**: For the structure of the website.
- **CSS3**: For styling and responsive design.
- **JavaScript**: To power interactivity and integrate the particle effect.
- **particles.js**: For the animated particle background.
- **FontAwesome**: For scalable vector icons.
